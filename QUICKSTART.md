## First Time Use

### 1. Welcome Screen
- You'll see a welcome modal on first visit
- Click "Get Started" to begin

### 2. Load Demo Data (Optional but Recommended)
To understand how the app works:

## Daily Workflow

### Morning (1 minute)
1. Open **Dashboard**
2. Check "Today's Focus" for recommended topics
3. Note your current streak

### After Practice Session (2-3 minutes)
1. Go to **Daily Log**
2. Enter session details:
   - Date (defaults to today)
   - Time spent (use quick buttons: 30/60/90/120 min)
   - Check resources used and enter question counts
   - Select topics covered (Ctrl/Cmd + click for multiple)
   - Add optional notes
3. Click "Save Log"

### Weekly Review (5 minutes)
1. Check **Analytics** page
2. Review:
   - Weekly progress vs 14-hour goal
   - Topic coverage gaps (red topics need attention)
   - Practice time trends
   - Burnout insights
3. Adjust next week's focus accordingly

### When Applying to Jobs (2 minutes)
1. Go to **Applications**
2. Add new application:
   - Company name
   - Job title
   - Application link
   - Date applied
   - Status
   - Optional notes
3. Update status as you progress through interviews

## Common Questions

**Q: Can I use this on mobile?**
A: The app is desktop-optimized for evening study sessions. It will work on mobile but the experience is better on desktop.

**Q: What if I miss a day?**
A: Your streak resets, but all your data remains. Start a new streak tomorrow!

**Q: Can I export my data?**
A: Currently, data is stored in browser localStorage. You can't export, but you can screenshot analytics for interviews.

**Q: How do I backup my data?**
A: Data is stored in browser localStorage. To backup, use the same browser on the same device. Clearing browser data will reset the app.

**Q: Can I change the weekly goal?**
A: The 14-hour weekly goal is configured in settings. You can modify it in the code (`js/app.js`).

**Q: What happens if I use different browsers?**
A: Each browser has separate localStorage. Your data won't sync across browsers.
